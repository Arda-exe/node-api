Games API Documentation

Base URL: http://localhost:3000

This API provides endpoints to manage video games and their developers with full CRUD operations, filtering, sorting, and pagination capabilities.

Developers Endpoints

Method Endpoint Description Parameters
GET /api/developers Get all developers with game count None
GET /api/developers/:id Get a single developer by ID with game count id - Developer ID (path parameter)
GET /api/developers/:id/games Get all games by a specific developer id - Developer ID (path parameter)
POST /api/developers Create a new developer Request Body (JSON):
• name - Developer name (required, string)
• country - Country (required, string)
• founded_year - Founded year (required, number, 1950-current)
PUT /api/developers/:id Update an existing developer id - Developer ID (path parameter)
Request Body (JSON):
• name - Developer name (required, string)
• country - Country (required, string)
• founded_year - Founded year (required, number, 1950-current)
DELETE /api/developers/:id Delete a developer (cascades to games) id - Developer ID (path parameter)

Games Endpoints

Method Endpoint Description Parameters
GET /api/games Get all games with optional filtering, sorting, and pagination Query Parameters:
• limit - Number of results per page (default: 10)
• offset - Number of results to skip (default: 0)
• search - Search in title and description
• genre - Filter by single genre
• genres - Filter by multiple genres (comma-separated)
• sort - Sort field (rating, release_year, title, id)
• order - Sort direction (asc, desc, default: asc)
• minRating - Minimum rating (0-10)
• maxRating - Maximum rating (0-10)
GET /api/games/:id Get a single game by ID id - Game ID (path parameter)
POST /api/games Create a new game Request Body (JSON):
• title - Game title (required, string)
• description - Game description (required, string)
• genre - Genre (required, string)
• release_year - Release year (required, number, 1950-current)
• platform - Platform (required, string)
• rating - Rating (required, number, 0-10)
• developer_id - Developer ID (required, number)
PUT /api/games/:id Update an existing game id - Game ID (path parameter)
Request Body (JSON):
Same fields as POST (all required)
DELETE /api/games/:id Delete a game id - Game ID (path parameter)
