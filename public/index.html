Games API Documentation

Base URL: http://localhost:3000

Developers Endpoints

Method Endpoint Description Parameters
GET /api/developers Get all developers None
GET /api/developers/:id Get single developer by ID id (path parameter)
GET /api/developers/:id/games Get all games by a developer id (path parameter)
POST /api/developers Create a new developer Body: name, country, founded_year, website (optional)
PUT /api/developers/:id Update a developer id (path parameter), Body: name, country, founded_year, website
DELETE /api/developers/:id Delete a developer id (path parameter)

Games Endpoints

Method Endpoint Description Parameters
GET /api/games Get all games with filtering, sorting, and pagination Query Parameters:
- limit (default: 10)
- offset (default: 0)
- search (searches title and description)
- genre (filter by single genre)
- genres (filter by multiple genres, comma-separated)
- sort (rating, release_year, price, title)
- order (asc or desc)
- minPrice, maxPrice (price range)
- minRating, maxRating (rating range 0-10)
GET /api/games/:id Get single game by ID id (path parameter)
POST /api/games Create a new game Body: title, description, genre, release_year, platform, rating (0-10), price, developer_id
PUT /api/games/:id Update a game id (path parameter), Body: title, description, genre, release_year, platform, rating, price, developer_id
DELETE /api/games/:id Delete a game id (path parameter)
